# Semantic Profile Page

A clean, professional profile page built with semantic HTML5. Includes sections for About, Skills, Projects, and a Contact form.

## Features
- Semantic structure: header, nav, main, sections, articles, footer
- Jump-link navigation to page sections
- Skills list
- Projects section with article elements
- Contact form and connected labels
- Responsive layout and profile image
